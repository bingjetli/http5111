About Me

Born in the tropical Caribbean country of Belize, I’ve been interested in software development ever since I was a teenager in high school.

I started off learning how to make flash games with Actionscript 2.0, later transitioning into the C-like scripting language known as Pawn. From there I branched off into several other programming languages such as C, C++, Java, Python, Javascript and so on.

I have a growing interest in machine learning and artificial intelligence, particularly when it comes to computer vision and generative AI.

Outside of software development, I also like to optimize for fitness, health and personal development.

Here's a quick overview of my notable projects!

VU Restaurant Reservation System

A full-stack restaurant reservation system created for VU Restaurants. The frontend is powered by React-Native and the Expo-CLI. The backend is a REST-API built using Node.JS, Express.JS and MongoDB.

International Height Visualizer

A visual comparison of the average heights of both males and females in each country using a Kaggle dataset. This project uses Node.JS, Express.JS and JQuery.

Modal Reader Chromium Extension

A chromium extension I built to help me read large blocks of programming documentation. It uses principles of typography and design to optimize the overall reading experience.

More details can be found on my Portfolio!
